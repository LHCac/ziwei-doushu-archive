# 一、納音五行屬水｜斗數水二局

| 甲子組 | 納音 | 性質摘要 |
|---|---|---|
| 丙子、丁丑 | 澗下水 | 有定性但無定向，源頭之水，深淺難測 |
| 甲寅、乙卯 | 大溪水 | 剛強不服輸，水勢強勁，易忙中有錯 |
| 壬辰、癸巳 | 長流水 | 源遠流長，多情卻似無情，遊戲人生感 |

## 2.1 丙子、丁丑｜澗下水

### 象意

澗下水是江河源頭，匯集細流而成。水勢可大，但流向不固定，水質澄清，深淺難測。

### 人格傾向

```text
有定性，無定向。
```

有自己的內在穩定性，但方向感容易變動，可能時向東、時向西。

### 最忌

```text
庚午、辛未：路傍土
戊申、己酉：大驛土
戊寅、己卯：城頭土
```

## 八、實戰使用方式

### 1. 排盤時看五行局

斗數五局可用來理解命盤起運與命宮定局的基礎：

```text
水二局、木三局、金四局、土五局、火六局
```

此檔主要用來查「某甲子納音屬什麼五行局」。

### 2. 解讀時不要單用納音斷命

納音五行只能作為輔助背景，不能單獨斷定個性或命運。實戰仍要合參：

```text
命宮主星
身宮
三方四正
四化
祿羊陀
大限
流年
實際問題背景
```

### 3. 數字喜忌僅作參考

原文提供各納音的喜忌數字，適合作為作者系統中的輔助參考，不宜直接當成絕對吉凶。
