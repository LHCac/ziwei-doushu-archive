# 四化快速查表｜化祿、化權、化科、化忌

> 來源：整理自文章作者補充〈四化〉，原文標示日期 2013-04-09。  
> 用途：作為四化判讀時的快速查表。  
> 注意：此表偏向作者系統與飛星／宮干四化觀點，實戰時仍需搭配本命盤、大限、流年、宮位、主星與三方四正。

---

## 1. 四化一句話總表

| 四化 | 一句話 | 核心關鍵字 |
|---|---|---|
| 化祿 | 有情、有緣、有資源 | 喜歡、福氣、財、緣分、貴人、鍾愛 |
| 化權 | 想掌控、想推動、想改變 | 權力、管束、成就、行動、原則、壓力 |
| 化科 | 被看見、被肯定、重形象 | 名聲、文書、考試、貴人、體面、教化 |
| 化忌 | 放不下、卡住、牽掛深 | 執著、虧欠、是非、耗損、糾纏、壓力 |

---

## 2. 四化白話記憶法

```text
祿：我喜歡，我有情，我有資源。
權：我要管，我要推動，我要掌控。
科：我被看見，我有名聲，我要好看。
忌：我放不下，我卡住，我欠它。
```

## 3. 化祿快速查表

### 基本意思

| 面向 | 解讀 |
|---|---|
| 人際 | 有情、有緣、喜歡、靠近 |
| 財務 | 有資源、有財氣，但不一定守得住 |
| 心理 | 舒服、喜愛、想享受 |
| 六親 | 有感情、鍾愛、牽掛 |
| 事業 | 有助力、有機會、有合作緣 |
| 風險 | 太多情、太享受、不專心、祿被忌帶走 |

### 判讀口訣

```text
看到化祿，先問：
這個人喜歡什麼？
資源在哪裡？
緣分在哪裡？
有沒有被忌帶走？
```

## 4. 化權快速查表

### 基本意思

| 面向 | 解讀 |
|---|---|
| 人際 | 管束、要求、控制、主導 |
| 財務 | 想掌控錢，理財有方法，但辛苦 |
| 心理 | 強勢、有原則、想改變現況 |
| 六親 | 對該宮人物有要求或管束 |
| 事業 | 有權力、有責任、有行動力 |
| 風險 | 太霸道、太硬、表達錯誤、權忌交戰 |

### 判讀口訣

```text
看到化權，先問：
誰想掌控？
哪裡要負責？
哪裡有壓力？
權力用得對不對？
```

## 5. 化科快速查表

### 基本意思

| 面向 | 解讀 |
|---|---|
| 人際 | 貴人、體面、文質彬彬 |
| 財務 | 名聲帶財、專業形象帶資源 |
| 心理 | 愛美、要面子、重名聲 |
| 六親 | 該宮人物像貴人，或形象較好 |
| 事業 | 考試、文書、作品、曝光、名聲 |
| 風險 | 虛名、表面好看、硬撐形象 |

### 判讀口訣

```text
看到化科，先問：
哪裡被看見？
哪裡有貴人？
哪裡需要形象？
是真的有內涵，還是只有表面？
```

## 6. 化忌快速查表

### 基本意思

| 面向 | 解讀 |
|---|---|
| 人際 | 虧欠、放不下、關心過度 |
| 財務 | 耗損、卡住、壓力、執著 |
| 心理 | 牽掛、焦慮、鑽牛角尖 |
| 六親 | 先天債、虧欠、深層牽連 |
| 事業 | 工作壓力、深耕、專業累積 |
| 風險 | 是非、耗損、糾纏、情緒壓力 |

### 判讀口訣

```text
看到化忌，先問：
命主放不下什麼？
卡在哪裡？
欠在哪裡？
能不能把執著變成專業？
```

## 7. 生年四化 vs 命宮干四化

| 類型 | 意義 | 白話理解 |
|---|---|---|
| 生年四化 | 先天、基礎、實質、天生帶來 | 你天生帶什麼氣 |
| 命宮干四化 | 後天、自身作為、努力、對待關係 | 你後天怎麼使用這些氣 |

記憶法：

```text
生年四化：命盤原本給你的設定。
命宮干四化：你自己怎麼跟世界互動。
```

## 9. 生年三吉的判讀

三吉指：

```text
化祿、化權、化科
```

### 三吉在哪裡比較有力？

| 位置 | 解讀 |
|---|---|
| 三吉守命 | 靜態、本有、命好，但未必有發揮 |
| 三吉在財、官、遷 | 動態、有舞台、有發揮空間 |
| 三吉照流年命、財、官 | 該年較容易啟動成果 |
| 三吉入大限三合 | 該大限較能發揮先天優勢 |

口訣：

```text
命為體，財官遷為用。
三吉入用，才比較有活力。
```

## 10. 自化快速查表

### 自化基本觀念

```text
某宮有生年四化。
該宮宮干又自化。
代表這股力量被自己化出去，通常會牽動對宮。
```

### 生祿在宮自化

| 自化 | 快速解讀 |
|---|---|
| 自化祿 | 祿出，輕鬆得、輕鬆花 |
| 自化權 | 有計畫、有節制地使用資源 |
| 自化科 | 炫耀、虛華、重表面 |
| 自化忌 | 祿隨忌走，資源流失感強 |

### 生權在宮自化

| 自化 | 快速解讀 |
|---|---|
| 自化祿 | 權被帶到對宮，外面有貴人或發揮空間 |
| 自化權 | 權力抵銷，空有權影 |
| 自化科 | 有貴人牽引，但不一定帶財 |
| 自化忌 | 權忌交戰，執著、矛盾、表達錯誤 |

### 生科在宮自化

| 自化 | 快速解讀 |
|---|---|
| 自化祿 | 有貴人賞識，才學可發揮 |
| 自化權 | 能展現所長 |
| 自化科 | 科出，表面好看但貴人力不足 |
| 自化忌 | 形象被破壞，仍苦追表面美 |

### 生忌在宮自化

| 自化 | 快速解讀 |
|---|---|
| 自化祿 | 雙忌，壓力在對宮 |
| 自化權 | 霸道、逞強、不擇手段 |
| 自化科 | 貴人無力，硬撐場面 |
| 自化忌 | 雙忌直沖對宮，對宮壓力重 |

## 11. 祿忌關係

### 核心口訣

```text
祿是因，忌是果。
```

也就是：

| 祿 | 忌 |
|---|---|
| 起因 | 結果 |
| 喜歡 | 執著 |
| 有情 | 放不下 |
| 想要 | 卡住 |
| 資源 | 耗損 |

### 祿忌特別要注意的情況

| 組合 | 解讀 |
|---|---|
| 祿忌同宮 | 喜歡與壓力在同一處，容易糾纏 |
| 祿忌一六宮位 | 視同祿忌同宮，祿容易隨忌走 |
| 祿忌三合 | 祿忌互相牽動，有吉凶混合 |
| 祿被忌帶走 | 得到後留不住，或因有情而生苦 |

### 判讀問題

```text
1. 祿在哪裡？命主想要什麼？
2. 忌在哪裡？最後卡在哪裡？
3. 祿忌是否同宮、一六、三合？
4. 是否形成祿隨忌走？
5. 對宮是否被沖？
6. 是否有自化？
```

## 12. 命宮干化忌入十二宮速查

| 命忌入宮 | 快速解讀 |
|---|---|
| 命忌入父母 | 與父母、長輩、文書、表達有壓力；易出口傷人 |
| 命忌入兄弟 | 手足同儕緣較淺，朋友交不深 |
| 命忌入夫妻 | 對配偶不放心，感情安全感不足 |
| 命忌入福德 | 精神壓力、財務焦慮、配偶較累 |
| 命忌入子女 | 管子女、損財、桃花、外出、不喜待家 |
| 命忌入田宅 | 喜在家，適合室內或居家工作 |
| 命忌入財帛 | 敢花錢，錢直直付出去，不一定是守財奴 |
| 命忌入官祿 | 適合上班族，對工作執著 |
| 命忌入交友 | 與朋友、同事、部屬合不來 |
| 命忌入遷移 | 喜外出、流浪、心在外面 |
| 命忌入疾厄 | 說話直、易招小人、文書麻煩、與父母緣薄 |

## 14. 四化實戰判讀順序

分析四化時，建議照這個順序：

```text
1. 先分清楚是哪一種四化
   - 生年四化
   - 命宮干四化
   - 大限四化
   - 流年四化
   - 流月四化

2. 看化入哪一宮
   - 命、財、官、遷比較容易外顯
   - 六親宮要用關係語言解讀
   - 福德、疾厄要看心理與身體壓力

3. 看該宮主星本質
   - 例如天機化權與武曲化權，不可同論

4. 看是否自化
   - 自化會把力量帶出，常牽動對宮

5. 看祿忌關係
   - 同宮、一六、三合都要特別注意

6. 看三方四正與煞曜
   - 吉星加強，煞忌增加壓力

7. 最後回到問題本身
   - 感情、事業、財務、健康、學業、家庭，要用不同語言解讀
```

## 15. 解盤時的現代化說法

| 傳統說法 | 建議改成 |
|---|---|
| 忌守身命，一生不順 | 人生某些議題較有執著與壓力，需要長期修正 |
| 雙忌大凶 | 對宮或相關宮位壓力很重，要保守處理 |
| 祿被忌帶走 | 資源留不住，容易因喜歡而耗損 |
| 權忌交戰 | 想掌控但卡住，容易表達錯誤或用力過猛 |
| 科出無貴人 | 表面好看，但實質支援不足 |
| 生忌入六親為先天債 | 對該宮人物有深層牽掛、責任或放不下 |

## 16. 最重要的十句話

```text
1. 祿不是只有錢，也代表情、緣、福、資源。
2. 權不是只有權力，也代表管束、掌控與推動。
3. 科不是只有考試，也代表名聲、貴人與形象。
4. 忌不是只有倒楣，也代表執著、虧欠與深入。
5. 生年四化看先天，命宮干四化看後天對待。
6. 三吉入命是本有，三吉入財官遷才容易發揮。
7. 六親宮看四化，要用情、管、貴人、虧欠來看。
8. 自化會把原本力量化出去，通常牽動對宮。
9. 祿是因，忌是果。
10. 有忌的人，只要把執著用在專業，也能苦中求成。
```
