## 1. 使用提醒

這份資料主要是「小星、雜曜、流曜」的快速查表，適合用來補充細節，不適合作為主要解盤依據。

解盤優先順序仍建議：

```text
命宮主星 → 三方四正 → 四化 → 祿羊陀 → 大限 → 流年 → 小星雜曜補充
```

小星的功能比較像「修飾語」：

```text
主星決定主題。
四化決定事件動能。
小星補充細節、氣氛、提醒與觸發點。
```

## 1.2 乙級星分類理解

### 貴人、貴氣類

```text
天貴、恩光、天官、鳳閣、龍池、天德、月德、解神、天福、天壽
```

重點：

- 有貴人。
- 有貴氣。
- 有保護力。
- 逢凶化吉。
- 事情較容易有人幫忙或轉圜。

---

### 領導、輔佐、圓融類

```text
三台、八座、台輔、封誥
```

重點：

- 處事較圓融。
- 有應變能力。
- 有組織與領導感。
- 遇事比較能應付。
- 適合擔任協調、管理、幕僚或輔助角色。

---

### 才智、研究類

```text
天才、天巫
```

重點：

- 天才：聰明、機警、反應快、有才幹。
- 天巫：喜推究事理，對研究、技術、升遷、事業有幫助。

---

### 桃花、人緣、外貌類

```text
天喜、咸池
```

重點：

- 桃花。
- 異性緣。
- 容貌佳。
- 人際吸引力。

實戰提醒：桃花星要看是否與煞忌同宮或會照；吉則人緣、喜慶、美感，凶則感情困擾或人際糾纏。

---

### 孤獨、煩惱、虛耗類

```text
孤辰、寡宿、天哭、天虛、破碎、陰煞、蜚廉、天月
```

重點：

- 孤僻或不易親近。
- 容易心煩、挫折。
- 小人、口舌或暗中困擾。
- 事情有波動或不順。
- 身心狀態需照顧。

---

### 宗教、哲理、玄學類

```text
華蓋
```

重點：

- 清高。
- 喜哲理、宗教、玄學。
- 有孤高氣質。
- 適合學習命理、宗教、藝術與精神性知識。

```text
天德、將星、龍德、攀鞍
```

重點：

### 流年耗損與不順類

### 流年桃花、精神、移動類

```text

```

重點：

- 咸池：桃花、人緣、異性緣。
- 華蓋：宗教、哲理、玄學、清高。
- 歲驛：遷動多、忙碌、外出、變動。

---

### 歲建

歲建代表流年吉凶的重要觸發點。

原文說：

```text
歲建代表流年吉凶，會吉則吉；會煞則比較不吉。
```

實戰可理解為：

> 歲建像流年的主氣。它本身不一定單獨決定吉凶，要看它會到哪些主星、四化、吉曜、煞曜。

## 4. 實戰使用方式

### 4.1 命宮看乙級星

乙級星在命宮時，可以補充個性與氣質。

例如：

```text
命宮有天才：反應快、機警。
命宮有華蓋：清高，對宗教哲理有興趣。
命宮有孤辰、寡宿：個性較孤僻，關係距離感較強。
命宮有天貴、恩光：有貴氣或貴人緣。
```

---

### 4.2 流年命宮看丙級、丁戊級星

流年命宮見小星時，可以作年度氣氛參考。

例如：

```text
流年命宮見大耗：該年花費較大，要注意預算。
流年命宮見病符：該年健康與作息要注意。
流年命宮見歲驛：該年忙碌、移動、出差、變動多。
流年命宮見咸池：該年人緣、桃花、美感與社交增加。
流年命宮見華蓋：該年可能對宗教、玄學、哲理、藝術較有興趣。
```

---

### 4.3 小星不能單獨斷事

不可只看到一顆小星就斷吉凶。

例如：

```text
看到白虎，不可直接斷災。
看到咸池，不可直接斷外遇。
看到大耗，不可直接斷破財嚴重。
看到病符，不可直接斷疾病。
```

應改成：

```text
白虎：安全、衝突、工具、交通要小心。
咸池：社交、桃花、人際吸引增加，也需注意界線。
大耗：花費增加，預算需保守。
病符：健康、作息、壓力需管理。
```

## 5. 小星現代化說法對照

| 傳統說法 | 現代化說法 |
|---|---|
| 災厄、不順利 | 風險提高，需保守與檢查細節 |
| 喪服 | 情緒低氣壓、家人長輩議題、悲傷事件提醒 |
| 疾病 | 健康與作息管理提醒，不作醫學診斷 |
| 桃花多、風流 | 人緣、吸引力、社交增加，也需注意關係界線 |
| 耗盡、耗損 | 花費、資源流失、預算壓力 |
| 受牽制、束縛 | 行動受限制，需處理規則、合約或責任 |
| 被說閒話 | 注意溝通、形象與社群／職場流言 |
